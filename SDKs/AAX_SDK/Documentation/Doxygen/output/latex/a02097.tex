\doxysubsection{Description}
Cross-\/platform interface for access to the host\textquotesingle{}s direct memory access (DMA) facilities. 

\begin{DoxyRefDesc}{\+:\+Implemented by the AAX Host}
\item[\mbox{\hyperlink{a00805__aax_host_implementation000005}{\+:\+Implemented by the AAX Host}}]\end{DoxyRefDesc}


This interface is provided via a DMA port in the plug-\/in\textquotesingle{}s algorithm context.
