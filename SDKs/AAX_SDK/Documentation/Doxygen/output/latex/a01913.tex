Registers one or more algorithm processing entrypoints (process procedures) 

Any non-\/overlapping set of processing entrypoints may be specified. Typically this can be used to specify both Native and TI entrypoints using the same call.

The AAX Library implementation of this method includes backwards compatibility logic to complete the Process\+Proc registration on hosts which do not support this method. Therefore plug-\/in code may use this single registration routine instead of separate calls to \mbox{\hyperlink{a01905_a4acfd619cfbd35ffd0ff20ca67bda305}{Add\+Process\+Proc\+\_\+\+Native()}}, \mbox{\hyperlink{a01905_adf549e7f348c670b79bdb5f283798db2}{Add\+Process\+Proc\+\_\+\+TI()}}, etc. regardless of the host version.
