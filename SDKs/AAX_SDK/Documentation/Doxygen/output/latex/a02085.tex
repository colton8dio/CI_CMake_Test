\doxysubsection{Description}
Defines the display behavior for a parameter. 

This interface represents a delegate class to be used in conjunction with \mbox{\hyperlink{a02145}{AAX\+\_\+\+IParameter}}. \mbox{\hyperlink{a02145}{AAX\+\_\+\+IParameter}} delegates all conversion operations between strings and real parameter values to classes that meet this interface. You can think of \mbox{\hyperlink{a02169}{AAX\+\_\+\+ITaper\+Delegate}} subclasses as simple string serialization routines that enable a specific string conversions for an arbitrary parameter.

For more information about how parameter delegates operate, see the \mbox{\hyperlink{a02169}{AAX\+\_\+\+ITaper\+Delegate}} and \mbox{\hyperlink{a00829}{Parameter Manager}} documentation.

\begin{DoxyNote}{Note}
This class is {\itshape not} part of the AAX ABI and must not be passed between the plug-\/in and the host. 
\end{DoxyNote}
